# Publichook

Publichook is a passion project that I'm doing as of 6/21/2026 for fun because I have nothing better to do.
The script is AI-assisted / "vibe-coded"

## Usage
To load the script, simply load this loadstring into your executor of choice
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/CalTheSkid/Publichook/main/loader.lua?nocache="..tostring(os.time())..tostring(math.random(100000, 999999))))()
```

Feel free to copy any source code and use it in your own project!
